Selamat Datang

Ini adalah contoh website dalam 1 file HTML saja yang bisa dihosting di GitHub Pages.

Cara menggunakannya:

  1. Salin semua kode ini
  2. Buat file index.html di repository GitHub Anda
  3. Tempelkan kode ini
  4. Commit dan push ke GitHub

Tentang Kami

Website ini dibuat untuk menunjukkan bahwa Anda bisa membuat website sederhana dengan hanya 1 file HTML.

Semua CSS dan JavaScript sudah termasuk dalam file ini.
